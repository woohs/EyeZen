# 瞳憩 (EyeZen)

> 给眼睛片刻留白。

**瞳憩** 是一款极简、优雅的桌面护眼伴侣。它运行在系统的静谧角落，采用科学的「30分钟循环」法则，通过全屏深色沉浸引导，帮您从繁杂的屏幕中抽离。配以精美的可视化数据看板，让每一次微小休息，都成为养成健康用眼习惯的基石。

## 环境

- Python 3.10+
- 依赖见 `requirements.txt`：
  - `PySide6`（强大且现代化的 Qt UI 框架）

## 运行

```bash
pip install -r requirements.txt
python main.py
```

## 托盘与退出

- 托盘右键菜单实时显示**「倒计时」**进度。
- 首页可自定义**工作时间**，仅在该时间段内触发自动提醒；非工作时间会自动暂停。
- 首页支持配置**开机自动启动**，且默认开启；应用会写入 Windows 当前用户启动项，并以托盘驻留方式启动。
- 点击主窗口右上角 **关闭**：最小化到 **系统托盘**（主窗口隐藏，程序仍在运行）。
- 托盘图标 **左键**：打开主窗口（默认动作）。
- 托盘图标 **右键** 菜单：**打开主控制台**、**退出**（仅此处为真正退出程序）。
- 统计看板内可查看累计/今日/近 7 天图表数据与最近日志。数据保存在 `%LOCALAPPDATA%\EyeRest\stats.json`。

## 打包为 exe（可选）

```bash
pip install pyinstaller
pyinstaller --noconfirm --windowed --onefile --name EyeZen main.py
```

生成的可执行文件在 `dist` 目录。`--windowed` 可避免弹出控制台黑框。若打包后托盘不显示，可尝试去掉 `--onefile`，或根据 PyInstaller 提示增加 `--hidden-import`（例如 `pystray._win32`）。

## 说明

- 含主显示器的屏幕上显示提示文案与「结束本次休息」；其余屏幕为纯黑覆盖。
- 若多屏位置或缩放异常，请确认系统「缩放与布局」设置；程序已尝试启用 Windows 每显示器 DPI 感知。
